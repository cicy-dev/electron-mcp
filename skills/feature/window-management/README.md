# Window Management Skill

窗口管理功能测试集合。

## 测试文件

- `api.window-tools.test.js` - 窗口工具 API 测试
- `api.set-window-bounds.test.js` - 窗口边界设置 API 测试
- `window-tools.test.js` - 窗口工具测试
- `set-window-bounds.test.js` - 窗口边界设置测试

## 运行测试

```bash
cd skills/window-management
npm test
```

## 功能覆盖

- 打开/关闭窗口
- 获取窗口信息
- 设置窗口位置和大小
- 窗口生命周期管理
